最終的にはシステム上でいいかんじにシフトを組んでくれるところまで実装していく構想があった
実際には紙で掲示されたシフト表のデータを自分が直接データベースに登録する形で運用

REST API Perl

借りていたレンタルサーバで動かせる言語をいろいろ試していた時期のためPerlでの実装となっている
実装済みのエンドポイントの一覧は下記のとおり

特定の日付のシフトデータを取得する

GET /api/v1/shifts/{日付数字8ケタ}

特定の従業員のシフトデータを取得する

GET /api/v1/employees/{従業員番号}/shifts?since={日付数字8ケタ}&until={日付数字8ケタ}

従業員を登録する

POST /api/v1/employees
