で、結局どのプログラミング言語を勉強すればいいのか

2017-11-25

プログラミングの勉強を始めるとき、まずはどの言語を勉強するかを決めると思います。

初心者におすすめの言語を調べると、Java、Python、C#などの言語が書かれたページが出てきます。

それぞれの言語の特徴やメリットなどとともに、プログラミング言語の名前がたくさん並んだWebページ。

あなたはそれを読み終えて、こう思いませんでしたか？

「で、結局どの言語が良いの？」

このページでは、これからプログラミングを始める方が学ぶべき言語を、筆者の経験をもとに1つに厳選してお教えします。

運命の選択ではない

プログラミング言語の違いというと、あなたはものすごく大きな違いを想像しているかもしれません。

言語と言えば、日本語の”こんにちは”は、英語では”Hello”、中国語では”你ニー好ハオ”ですね。

使われている文字も発音も違うので、英語も中国語も両方使えるようになるのは難しそうです。

では、プログラミングはどうかというと、言語によって書き方が大きく異なるということはありません。

言語ごとのプログラムの書き方の違いは、日本の方言ぐらいの違いだと思ってもらえると良いと思います。

ある地方では「ええ天気やで」と言うけど、ある地方では「よか天気ばい」と言う。その程度の違いです。

実際に、異なる2つの言語で書かれたプログラムを見比べてみましょう。

同じことをするプログラムを、Javaという言語と、Visual Basicという言語で書くとこうなります。

int weather;
String result;

weather = getWeather();
if (weather == Weathers.SUNNY) {
    result = "良い";
}
else {
    result = "良くない";
}

System.out.println("今日は天気が" + result + "です。");

Dim weather As Integer
Dim result As String

weather = GetWeather()
If weather = Weathers.Sunny Then
    result = "良い"
Else
    result = "良くない"
End If

Console.WriteLine("今日は天気が" & result & "です。")

書いてあることの意味はわからなくても、書き方が少し違うだけというのはわかっていただけたかと思います。

つまり、1つの言語を習得した後に、もう1つ言語を覚えるのは難しくないということです。

プログラミング言語の選択は確かに重要なことかもしれませんが、迷いすぎる必要はありません。

筆者はJavaを選択、しかし

筆者が最初に勉強することに決めたプログラミング言語はJavaです。

理由は単純で、筆者は当時スマートフォンのアプリを作りたいと思っていたからです。

Javaは、Androidのアプリケーション開発に利用されています。

さて、Javaの勉強を進めて、自分のアプリを開発することはできましたが、2つの問題に直面しました。

1つは、iPhoneを使っている友人に、自分が作ったアプリを使ってもらうことができなかったことです。

Javaで作ることができるのはAndroidアプリです。当然ですが、iPhoneで使うことはできません。

せっかく作ったアプリでしたが、Androidを使っていた少人数の方にしか入れてもらえませんでした。

そして、もう1つの問題はというと、インターネットに接続して何かをする機能を作れなかったことです。

作れなかった機能はどんなものかというと、例えばゲームの得点ランキングを表示する機能です。

サーバーという言葉はどこかで聞いたことがあると思います。

ランキングを表示するためには、サーバーにゲームの得点の情報を送る必要があります。

この時、サーバーでもプログラムを動かすのですが、サーバーでJavaのプログラムを動作させるのは簡単なことではなかったのです。

Javaしか勉強していない段階だったので、インターネットに接続する機能は諦めていました。
